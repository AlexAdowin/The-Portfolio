Mes Compétences Back-End

Langages de Programmation

PHP

PHP est mon langage de prédilection pour le développement web côté serveur. Il me permet de créer des sites web dynamiques et interactifs tout en intégrant des bases de données.

  • Création de sites dynamiques avec PHP et MySQL
  • Utilisation de la programmation orientée objet (POO)
  • Gestion de la sécurité des applications web (XSS, CSRF, etc.)

Node.js

Node.js permet d'exécuter JavaScript côté serveur. Il est idéal pour créer des applications web rapides et évolutives avec un grand nombre de connexions simultanées.

  • Création d'APIs RESTful avec Express.js
  • Gestion d'événements et des flux en temps réel (WebSockets)
  • Utilisation des modules Node.js pour l'authentification et le stockage de fichiers

Systèmes de Gestion de Base de Données

MySQL

MySQL est ma solution de choix pour la gestion de bases de données relationnelles. Je m'assure d'utiliser des requêtes optimisées et sécurisées pour garantir les performances des applications.

  • Création et gestion de bases de données relationnelles
  • Optimisation des requêtes SQL pour des performances accrues
  • Sécurisation des données avec des requêtes préparées

MongoDB

MongoDB est une base de données NoSQL que j'utilise pour des applications nécessitant une grande flexibilité de stockage de données non structurées.

  • Création de collections et documents dans MongoDB
  • Gestion de bases de données NoSQL avec Mongoose
  • Optimisation des performances et des requêtes avec MongoDB

API et Web Services

APIs RESTful

Les APIs RESTful permettent d'offrir des services web qui communiquent efficacement entre différents systèmes. Je suis capable de créer des APIs robustes et sécurisées avec Node.js et PHP.

  • Conception d'APIs RESTful pour la gestion des ressources
  • Utilisation des méthodes HTTP (GET, POST, PUT, DELETE)
  • Authentification et autorisation via JWT

GraphQL

GraphQL est une alternative moderne à REST pour interroger des données de manière plus efficace. Je l'utilise pour des applications nécessitant des requêtes flexibles et spécifiques.

  • Conception de schémas GraphQL
  • Gestion des résolveurs pour obtenir des données précises
  • Intégration avec des bases de données SQL et NoSQL

Gestion des Sessions et Authentification

JWT (JSON Web Tokens)

JWT est une méthode d'authentification moderne que j'utilise pour sécuriser les sessions des utilisateurs dans les applications web et mobiles.

  • Création et gestion des tokens d'authentification
  • Validation des tokens pour sécuriser les API
  • Gestion des expirations de session et des rafraîchissements de token

OAuth

OAuth est un protocole standard d'autorisation. Je l'utilise pour permettre aux utilisateurs de se connecter via leurs comptes Google, Facebook, etc., tout en respectant les bonnes pratiques de sécurité.

  • Implémentation de l'authentification OAuth 2.0
  • Utilisation des flux d'autorisation pour des connexions sécurisées
  • Intégration avec des services tiers pour la gestion des accès

Outils et Environnements de Développement

Docker

Docker me permet de créer des environnements isolés et reproductibles pour le développement et le déploiement d'applications backend, garantissant ainsi une meilleure gestion des dépendances et des versions.

  • Création de conteneurs pour l'exécution des applications
  • Utilisation de Docker Compose pour gérer plusieurs services
  • Optimisation des environnements de développement et de production

NGINX

NGINX est un serveur web que j'utilise pour servir les applications backend, gérer les requêtes HTTP et améliorer les performances de l'application en tant que reverse proxy.

  • Configuration de NGINX comme reverse proxy
  • Optimisation des performances de serveurs backend
  • Gestion de la sécurité et des en-têtes HTTP
